## Hardhat常用命令
安装：npm install --save-dev hardhat

创建Hardhat项目：npx hardhat

编译合约：npx hardhat compile

测试合约：npx hardhat test

部署合约：npx hardhat run scripts/deploy.js --network goerli

合约认证：npx hardhat verify address --network goerli
